# AIscentra

**Intelligence Observatory** — Observe. Analyze. Accelerate the Future.

AIscentra monitors the global AI ecosystem, detects meaningful signals, and transforms fragmented information into structured intelligence.

## Getting Started

```bash
# 1. Clone and install
git clone https://github.com/aiscentra/AIscentra.git
cd AIscentra
npm install

# 2. Configure environment
cp .env.example .env.local
# Fill in Supabase and OpenRouter credentials

# 3. Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Development

```bash
npm run dev          # Development server
npm run build        # Production build
npm run type-check   # TypeScript validation
npm run lint         # ESLint
npm run format       # Prettier
npm run validate     # All checks
```

## Project Foundation

This project is built on a complete strategic architecture:
- **Intelligence Observatory** concept — signals are the primary entity
- **Signal Scoring Specification** — three-dimensional scoring system
- **Intelligence Evolution Framework** — how knowledge compounds over time
- **Build Order** — 16-stage implementation sequence

## Stack

- **Frontend:** Next.js 15, TypeScript, Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Edge Functions, Auth)
- **AI:** OpenRouter (model-agnostic, budget-controlled)
- **Hosting:** Vercel
